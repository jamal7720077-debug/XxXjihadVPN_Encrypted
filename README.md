# XxXjihad VPN Manager (Encrypted Edition)

هذا المستودع يحتوي على النسخة المشفرة والآمنة من XxXjihad VPN Manager، مع جميع الإصلاحات والميزات الجديدة المطلوبة. تم تشفير جميع ملفات المكتبة (`.sh`) باستخدام AES-256-CBC ويتم فك تشفيرها وتنفيذها مباشرة في الذاكرة لضمان أقصى درجات الأمان وحماية الملكية الفكرية.

## الميزات الرئيسية:

*   **إصلاحات حرجة:** تم حل جميع أخطاء بناء الجملة في القوائم، ومشكلة SSH Banner، وضمان تحميل جميع الدوال بشكل صحيح، وتحديث روابط التحميل.
*   **أمان مطلق:**
    *   تشفير جميع ملفات `.sh` في `lib/` باستخدام AES-256-CBC.
    *   تنفيذ الكود مباشرة في الذاكرة (RAM) باستخدام `bash loader`، مما يمنع رؤية الكود المصدري على القرص.
    *   ميزة "Anti-Theft Lock" للتحقق من ملف `jihad` في `/etc/xxjihad/` لضمان عدم فصل الملفات.
*   **ميزات متقدمة:**
    *   **Auto SSL (Certbot):** خيار جديد لإصدار شهادات SSL تلقائيًا وتكوين HAProxy.
    *   **ألوان واجهة المستخدم الديناميكية:** عرض حالة الخدمة (نشطة/غير نشطة) بألوان خضراء/حمراء في قائمة البروتوكولات.
    *   **إدارة المستخدمين الاحترافية:**
        *   تحديد حصة استهلاك البيانات (GB) لكل مستخدم.
        *   حسابات تجريبية بانتهاء صلاحية تلقائي باستخدام أمر `at`.
        *   إنشاء جماعي للمستخدمين (50-100 مستخدم) بكلمات مرور تلقائية.
        *   عرض حالة الحساب (انتهاء الصلاحية، استهلاك البيانات، الحالة) في SSH banner.
*   **إلغاء تثبيت شامل:** خيار جديد (99) لإزالة جميع مكونات XxXjihad بشكل كامل، بما في ذلك الملفات والخدمات.
*   **توافق النظام:** دعم كامل لـ Ubuntu 22.04 و 24.04.

## كيفية التثبيت:

لتثبيت XxXjihad VPN Manager (النسخة المشفرة)، اتبع الخطوات التالية:

1.  **قم بتنزيل سكريبت التثبيت:**
    ```bash
    wget https://raw.githubusercontent.com/jamal7720077-debug/XxXjihadVPN_Encrypted/master/install.sh -O install.sh
    ```

2.  **اجعل السكريبت قابلاً للتنفيذ:**
    ```bash
    chmod +x install.sh
    ```

3.  **قم بتشغيل سكريبت التثبيت كـ root:**
    ```bash
    sudo ./install.sh
    ```

    سيقوم سكريبت التثبيت بتثبيت جميع التبعيات اللازمة، ونسخ الملفات المشفرة، وإعداد الـ `bash loader`، وإنشاء ملف قفل الحماية.

## كيفية الاستخدام:

بعد التثبيت الناجح، يمكنك تشغيل مدير XxXjihad VPN بكتابة الأمر التالي:

```bash
xxjihad
```

سيؤدي هذا إلى فتح القائمة الرئيسية حيث يمكنك إدارة المستخدمين والبروتوكولات والميزات الأخرى.

## ملاحظات هامة:

*   **لا تقم بتعديل الملفات المشفرة يدويًا.** أي تعديل قد يؤدي إلى تلف الكود ومنعه من العمل.
*   **حافظ على سرية مفتاح التشفير.** يتم تخزين المفتاح في `/etc/xxjihad/encryption.key` ويجب حمايته.
*   **الملكية الفكرية:** هذا الإصدار المشفر يهدف إلى حماية الكود المصدري الخاص بك. يرجى التأكد من أن استخدامك يتوافق مع جميع القوانين واللوائح المعمول بها.
